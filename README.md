# Bharatiya-Nyaya-Sanhita-2023-India-s-New-Criminal-Law
The Bharatiya Nyaya Sanhita, 2023 is India’s new criminal‑law code that replaces the Indian Penal Code, 1860, and consolidates and reorganises offences and punishments for crimes such as murder, theft, assault, terrorism‑related acts, and cyber‑crimes, with a focus on modernising and nationalising the substantive criminal law framework.
